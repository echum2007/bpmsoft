## Примеры применения

### Задача: Настроить Google Mail OAuth в BPMSoft

1. ✅ **Сначала:** `mcp__notebooklm__ask_question` → "Как настроить Google Mail OAuth 2.0?"
2. Если ответ неполный: `/bpmsoft-kb` → уточнение
3. Если сеть отключена: `Documentation 1.9/Для администратора/` → manual search
4. Для контекста: `knowledge/wiki/platform.md` → архитектура email

### Результат (тестовый запрос 2026-04-18)

NotebookLM ответил с полной информацией о OAuth в BPMSoft за **827 токенов** с цитатами:
- OAuth 2.0 работает только для Microsoft 365
- Для Gmail используется Basic auth (логин/пароль)
- IMAP/SMTP серверы: `imap.gmail.com:993`, `smtp.gmail.com:465`

## Техническое обслуживание

### Обновление NotebookLM блокнота

Если документация BPMSoft обновилась:
1. Открыть https://notebooklm.google.com
2. Загрузить новые PDF в блокнот `bpmsoft-documentation`
3. Перезагрузить MCP: `mcp__notebooklm__cleanup_data` + `mcp__notebooklm__re_auth`

### Отслеживание состояния

Команда для проверки здоровья:
```
mcp__notebooklm__get_health
```

Ожидается:
```json
{
  "status": "ok",
  "authenticated": true,
  "active_sessions": 0
}
```
