## How to Add Sources

1. Go to https://notebooklm.google.com/
2. Open each notebook from the list above
3. Click **"+ Add source"** button
4. Paste each URL from the corresponding section
5. Wait for NotebookLM to load the content
6. Repeat for all 5 notebooks
