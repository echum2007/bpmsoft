# Дополнительные получатели уведомлений (CC) по обращениям

В карточке обращения на закладке **Информация об обращении** появилось поле **CC (дополнительные получатели)**. Здесь можно указать адреса людей, которые будут автоматически получать копии всех уведомлений по обращению.

---

## Зачем это нужно

Раньше, чтобы держать кого-то в курсе по обращению, нужно было каждый раз вручную добавлять его в копию при ответе. Теперь достаточно один раз указать адрес в поле CC — и все уведомления по обращению будут уходить ему автоматически.

---

## Как заполнить поле CC

Откройте карточку обращения, перейдите на закладку **Информация об обращении** и найдите поле **CC (дополнительные получатели)**. Введите адреса через точку с запятой и сохраните запись:

```
ivan@example.com; maria@example.com
```

Если ввести адрес в неправильном формате, система предупредит об ошибке при сохранении.

---

## Как адреса попадают в поле CC автоматически

Когда клиент присылает письмо с кем-то в копии, система сама добавляет эти адреса в поле CC обращения. Если в следующем письме появятся новые адреса — они тоже добавятся. Повторов не будет.

---

## Два поля с похожим названием: в чём разница

При написании ответа клиенту в форме сообщения есть поле **Копия** — оно работает как обычное поле «Копия» в почтовом клиенте.

Это **не одно и то же**, что поле CC в карточке обращения:

- **Копия в форме ответа** — адрес получит только это одно сообщение. В следующих уведомлениях его уже не будет.
- **CC в карточке обращения** — адрес будет получать все уведомления по обращению, пока вы его не удалите.

Если нужно добавить человека в копию разово — используйте поле в форме ответа. Если нужно, чтобы он получал всё — добавьте его в карточку обращения.

Обратите внимание: если клиент ответит и укажет кого-то в копии своего письма, система автоматически занесёт этот адрес в поле CC карточки, и дальше уведомления будут уходить ему постоянно.

---

## CC в сервисном договоре

В карточке сервисного договора тоже есть поле CC. Адреса из него автоматически добавляются ко всем уведомлениям по обращениям в рамках этого договора.

Это удобно, если нужно постоянно держать в курсе, например, руководителя со стороны клиента — достаточно указать его адрес один раз в договоре, и он будет в копии по всем обращениям.

При отправке уведомления адреса из договора и из карточки обращения объединяются. Повторов не будет.

---

## Частые вопросы

**Поле CC пустое, хотя клиент писал с копией — почему?**
Адреса добавляются автоматически только при получении нового письма после внедрения функциональности. Для старых обращений поле нужно заполнить вручную.

**Можно ли убрать адрес из CC?**
Да — отредактируйте поле вручную и сохраните запись.
